# Standard Playground

Example of a standard JavaScript/TypeScript project using `@poupe/eslint-config`.

## Usage

```js
import { defineConfig } from '@poupe/eslint-config';

export default defineConfig({
  // Additional configuration
});
```

## Development

```bash
# Install dependencies
pnpm install

# Lint files
pnpm lint

# Fix lint issues
pnpm lint:fix
```
