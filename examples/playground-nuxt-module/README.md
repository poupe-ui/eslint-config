# Playground Nuxt Module

An example Nuxt module using `@poupe/eslint-config` for module development.

This module can be tested in the [playground-nuxt](../playground-nuxt) example.

## Development

```bash
# Install dependencies
pnpm install

# Generate type stubs
pnpm dev:prepare

# Lint the module
pnpm lint
```
